# Intent-Based Violence Prediction System

A Next.js landing page for an AI-powered violence prediction system.

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app` - Next.js app directory with layout and main page
- `/components` - React components for each section:
  - `Header.jsx` - Navigation header
  - `HeroSection.jsx` - Hero section with main headline
  - `AboutUsSection.jsx` - About us section with video preview
  - `ServicesSection.jsx` - Services overview
  - `HowItWorksSection.jsx` - Step-by-step process explanation
  - `TryPrototypeSection.jsx` - Demo/upload section
  - `BenefitsSection.jsx` - Benefits cards
  - `FooterSection.jsx` - Footer with contact information

## Technologies

- Next.js 14
- React 18
- Tailwind CSS
